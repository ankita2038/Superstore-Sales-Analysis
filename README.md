# Superstore Sales Analysis using Python

##  Project Overview

This project analyzes Superstore sales data using Python to identify sales trends, regional performance, customer segments, product performance, and business insights. The objective is to transform raw sales data into meaningful insights that support better business decisions.

---

#  Objectives

- Perform Data Cleaning and Preprocessing
- Analyze Key Performance Indicators (KPIs)
- Perform Exploratory Data Analysis (EDA)
- Analyze Sales Trends
- Analyze Regional Performance
- Identify Top & Bottom Selling Products
- Analyze Customer Segments
- Generate Business Insights
- Provide Business Recommendations

---

#  Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

Dataset: **Superstore Sales Dataset**

#  Key Findings

- Technology and Office Supplies contribute significantly to sales.
- West region generated the highest sales.
- Standard Class is the most preferred shipping mode.
- A small percentage of customers contribute a large share of total sales.
- Sales vary considerably across different states and cities.

---

#  Business Recommendations

1. Focus marketing campaigns on high-performing regions.
2. Reward loyal customers through loyalty programs.
3. Promote low-selling products using discounts.
4. Improve inventory planning based on monthly demand.
5. Optimize shipping operations for faster delivery.
